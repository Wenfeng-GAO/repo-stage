# Tiny CLI Tool

Tiny CLI Tool formats plain text task lists for terminal workflows.

It is useful for developers who keep small project checklists in Markdown and want a fast command to normalize indentation, remove extra blank lines, and print the result.

## Install

```bash
npm install -g tiny-cli-tool
```

## Usage

```bash
tiny-cli-tool format tasks.md
```

## Features

- Formats Markdown task lists.
- Removes repeated blank lines.
- Keeps completed and pending task markers unchanged.
