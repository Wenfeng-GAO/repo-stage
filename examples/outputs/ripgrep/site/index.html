Repository BurntSushi/ripgrep

ripgrep recursively searches directories for a regex pattern while respecting gitignore rules.

Problem

Searching source trees can produce noise from ignored, hidden, or binary files.

Solution

ripgrep recursively searches your current directory for a regex pattern.

Features

ripgrep respects gitignore rules and automatically skips hidden files and binary files.

ripgrep is exposed as the rg command for terminal search workflows.
