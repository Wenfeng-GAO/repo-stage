Node.js CLI framework

The complete solution for Node.js command-line interfaces.

Commander describes your CLI, parses arguments into options and command arguments, displays usage errors, and implements automated help.

Start with npm Browse examples
$ node split.js -s / --first a/b/c
[ 'a' ]
$ node string-util.js help split
Usage: string-util split [options] <string>

Quickstart

Describe the interface. Let Commander handle parsing and help.

These commands and API calls are grounded in the README quick start.

npm install commander
const { program } = require('commander');

program
  .option('--first')
  .option('-s, --separator <char>')
  .argument('<string>');

program.parse();

For growing CLIs

One set of parser primitives for small scripts and multi-command tools.

The README starts with a small `split.js` example and then moves to a multi-command `string-util.js` program. The generated page mirrors that path: begin with one command, then scale into named commands with descriptions and help.
