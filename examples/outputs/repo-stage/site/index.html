RepoStage

README in, project website out.

Start with the quickstart View on GitHub

Problem

Open-source maintainers may have useful repositories but weak external presentation.

Solution

RepoStage turns an existing open-source repo into a launch-ready project page.

Features

The workflow converts README, docs, package metadata, examples, license, and assets into repo-profile.json.

The required output includes site/index.html, site/styles.css, repo-profile.json, and README-gap-report.md.

Generated pages include hero, problem and solution, features, quickstart, examples, trust, and contribution sections when sourced.

Quickstart

Use the commands documented by the project

Step 1
repo-stage generate https://github.com/owner/repo --out site
