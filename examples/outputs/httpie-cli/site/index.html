Command-line HTTP client

Human-friendly HTTP for the API era.

HTTPie CLI helps developers test, debug, and interact with APIs and HTTP servers using simple syntax with formatted, colorized output.

Install HTTPie Try an example 
https httpie.io/hello

http PUT pie.dev/put \
  X-API-Token:123 \
  name=John

http --offline pie.dev/post hello=offline

Why HTTPie

Requests you can write, responses you can read.

The README describes HTTPie as a command-line client for creating and sending arbitrary HTTP requests. Its syntax is designed to feel natural, while the output is formatted and colorized for terminal reading.

API-first syntax

Send methods, headers, JSON data, forms, files, and arbitrary request data from concise terminal commands.

Readable output

Formatted and colorized output keeps response inspection approachable during testing and debugging.

Real workflows

HTTPS, proxies, authentication, custom headers, persistent sessions, and wget-like downloads are listed in the README features.
