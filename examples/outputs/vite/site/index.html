Repository vitejs/vite

Vite is a frontend build tool that provides a development server and optimized production builds.

Problem

Frontend projects need a quick development loop and a separate optimized production build.

Solution

Vite provides a development server and production build command for frontend projects.

Features

The README documents npm create vite@latest as a starting command.
